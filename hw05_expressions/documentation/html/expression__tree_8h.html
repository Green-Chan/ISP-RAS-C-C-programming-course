Reads an expression from string and forms a tree to represent it.

Parameters
[in]strC-string (terminated by '\0'), the string with expression (as described at main page)
[out]expr_treePointer to the tree (tree is always a pointer to the node structure)
[out]err_posThe pointer to the first incorrect symbol, if function finishes with an error, or the pointer to the next symbol after expression if function succeeds
Returns
0 on success, error code otherwise
Attention
expr_tree will be rewrited on success, so don't forget to call destruct_expression first if there was another tree there
