That is the function that performs the algorithm specified at the main page of the documentation. Sorts lines in text from file three times: in ascending order, in ascending order from the back of the line, to its original version.

Parameters
[in]file_in_pathPath to the file with text
[in]file_out_pathPath to the file where to write the resulting three versions
Attention
If file_out_path exists, it will be overwritten
Note
While comparing lines not alpha and not digit symbols are ignored, uppercase and lowercase symbols are considered equal.
The three resulting sections will be separated with lines, consisting from symbols '='.
