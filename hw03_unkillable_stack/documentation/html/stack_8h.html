Detailed Description

To use stack from this header you should define STACK_TYPE (the type of elements, stack will store), then include this file. If you want to use stacks with different types, undef STACK_TYPE after including this file, then define another STACK_TYPE and include this file again (repeat as many times as necessary).

To declare new stacks write TEMPLATE(type, stack) s1, s2; where type is the type of elements, stack will store. Always initialize stacks before using (use construct_stack) and destroy them when you don't need them (use destruct_stack).

To call stack method for some stack write <method_name>(type, ptr, [args]) where type is the type of elements, the stack stores, ptr is a pointer to the stack, and [args] are the other arguments of the method (if there are some).

See usage example in run_tests.cpp
