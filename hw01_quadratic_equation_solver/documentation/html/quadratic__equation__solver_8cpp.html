Solves the quadratic equation \( a x^2 + b x + c = 0 \)

Parameters
[in]aThe quadratic coefficient (coefficient a)
[in]bThe linear coefficient (coefficient b)
[in]cThe constant (coefficient c)
[out]first_rootReference to the first root
[out]second_rootReference to the second root
Returns
Number of roots
Note
If the number of roots is infinite, returns INF_ROOTS. If the function returns 1, the root is written in first_root. If the function returns 2, first_root is calculated by the formula \( \frac{-b + \sqrt{D}}{2a} \), second_root is calculated by formula \( \frac{-b + \sqrt{D}}{2a} \), where D is the discriminant.
