Compares the first word (the sequence of not space symbols to the end of file or first space symbol) at cur_symb with command or register at cmd. If they are equal, sets cur_symb to the next symbol after the first word.

Note
cmd should be static const char array from assembler.h so that sizeof(cmd) is the size of cmd
Parameters
[in]cmdString representation of command or register
[in,out]cur_symbPointer to the pointer to the current symbol of text being parsing
[in]file_endPointer to the symbol after last symbol of the text
Returns
1 if the first word is equal to the specified command or register, 0 otherwise

Compares the first word (the sequence of not space symbols to the end of file or first space symbol) at cur_symb with string representations of existing registers. If the first word equal to some of them, sets cur_symb to the next symbol after the word and returns the code of the corresponding register.

Parameters
