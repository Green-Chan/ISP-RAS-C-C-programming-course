void sort_text ( const char *  file_in_path,
const char *  file_out_sorted_path,
const char *  file_out_sorted_back_path,
const char *  file_out_origin_path,
language  lang 
)

That is the function that performs the algorithm specified at the main page of the documentation. Sorts lines in text from file three times: in ascending order, in ascending order from the back of the line, to its original version.

Parameters
[in]file_in_pathPath to the file with text
[in]file_out_sorted_pathPath to the file where to write the sorted version
[in]file_out_sorted_back_pathPath to the file where to write the sorted from back version
[in]file_out_origin_pathPath to the file where to write the origin version
[in]langThe language of the text
Attention
If file_out_path exists, it will be overwritten
Note
While comparing lines not alpha and not digit symbols are ignored, uppercase and lowercase symbols are considered equal. Only letters of the specified ( lang ) alphabet are not ignored.
